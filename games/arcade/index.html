Agent-built game testing hub

Agent Game Arcade

A launcher, scorecard, and smoke-test dashboard for games built by Merquery, DrClawBotNik, and future specialist agents.

Default test workflow

  1. Agent builds or updates a game under games/<id>/.
  2. Agent registers it in games/manifest.json.
  3. Run npm test to launch browser smoke tests.
  4. Review screenshots/videos in games/artifacts/.
  5. Eric plays the game and fills the manual scorecard.
