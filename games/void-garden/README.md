# Void Garden

A weird creative wildcard for the Agent Game Arcade: a cursor-haunted greenhouse where spores orbit the player, the moon-mouth blinks, and the specimen log mutates as you feed the garden.

## Controls

- Move mouse: lure spores and aim the cursor sigil.
- `F`: feed the garden, grow petals, and log specimens.
- `Space`: blink the moon-mouth and increase weirdness.
